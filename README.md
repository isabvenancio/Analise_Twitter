# 📱 Análise de Sentimentos em Tweets com NLP

Projeto de **Processamento de Linguagem Natural (NLP)** e **Machine Learning** para classificação de sentimentos em tweets utilizando o dataset **Sentiment140**.

## 📌 Funcionalidades

* Análise exploratória dos dados (EDA)
* Limpeza e pré-processamento dos textos
* Remoção de stopwords
* Visualização das palavras mais frequentes
* Geração de WordCloud
* Vetorização com **Bag of Words** e **TF-IDF**
* Treinamento de modelos de Machine Learning
* Avaliação com Accuracy, Precision, Recall, F1-Score e Matriz de Confusão
* Classificação de novos tweets

## 📥 Dataset

Baixe o dataset **Sentiment140** e coloque o arquivo **`training.1600000.processed.noemoticon.csv`** dentro da pasta **`data/`** antes de executar o projeto.

## 📚 Bibliotecas

Instale as dependências com:

```bash
pip install -r requirements.txt
```

## 🚀 Como executar

1. Clone este repositório.
2. Coloque o dataset na pasta `data/`.
3. Instale as dependências.
4. Abra e execute o notebook `notebooks/analise_sentimentos.ipynb`.
